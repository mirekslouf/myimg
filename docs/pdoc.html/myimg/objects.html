Module: myimg.objects

Key classes/objects for myimg package:

  1. MyImage class defines the basic MyImg object, which is used in most image manipulations.
  2. Montage class defines a montage/combination of several images, which are arranged in a rectangular grid/multi-image.
  3. Units, NumberWithUnits and ScaleWithUnits classes defines allowed units, number-with-units and scale-with-units, respectively.

Examples = how does it work?

MyImage class creates the basic object, which is used in most image manipulations within myimg.api module.

>>> # MyImage class :: simple example (short but real)
>>> import myimage.api as mi      # import API, which provides a simple UI
>>> img = mi.MyImage('some.bmp')  # open image: some.bmp
>>> img.label('a')                # insert a label in the upper left corner
>>> img.scalebar('rwi,100um')     # insert a scalebar to the lower right corner
>>> img.save_with_ext('_s.png')   # save the modified image to: some_ls.png

Montage class creates the multi-image object, which contains several images arranged in a rectangular grid.

>>> # Montage class :: simple example (short but real)
>>> import myimage.api as mi      # import API, which provides a simple UI
>>> images = ['s1.png','s2.png']  # define input images
>>> montage = mi.Montage(images,  # create montage image
>>>     itype='gray', grid=(1,2), # ...grayscale, just two images in a row
>>>     padding=10)               # ...padding/spacing between imgs = 10pixels
>>> montage.save('mreport.png')   # save the final montage of (the two) images

Units, NumberWithUnits, and ScaleWithUnits classes are used in myimg.utils.scalebar module when creating scalebars (as a scalebar contains number with units).

>>> # Units, NumberWithUnits and ScaleWithUnits classes :: typical/indirect use 
>>> # (the classes are employed behind the scenes when calling scalebar method
>>> img = mi.MyImage('some.bmp')  # open image: some.bmp
>>> img.scalebar('rwi,100um')     # scalebar (employs Units, NumberWithUnits... 
>>> img.save_with_ext('_s.png')   # save modified image: to some_s.png

>>> # NumberWithUnits class :: NON-typical/direct use
>>> # (the NumberWithUnits class can be used/tested for various conversions
>>> from myimg.objects import NumberWithUnits
>>> n =NumberWithUnits('0.1mm')
>>> print('Initial number_with_units:', n)  # prints 0.1 mm
>>> print('Setting correct units...')
>>> n.set_correct_units()
>>> print(n)                                # prints 100 um

 135    def get_font_size(self, font_name, required_font_size_in_pixels):
 136        '''
 137        Get font size (in fontsize units)
 138        corresponding to *required_font_size_in_pixels*.
 139
 140        Parameters
 141        ----------
 142        fontname : str
 143            Name of the TrueType font to use.
 144            Example (working in Windows): font_name='timesbd.ttf'
 145        required_font_size_in_pixels : float
 146            Required font size in pixels.
 147
 148        Returns
 149        -------
 150        font_size : int
 151            Final font size (in font units).
 152            If the returned *font_size* (in fontsize units)
 153            is applied to font with given *fontname*,
 154            then the height of the font (in pixels units)
 155            will correspond to *required_font_size_in_pixels* argument.
 156            
 157        Technical notes
 158        ---------------
 159        * This function is a modified recipe from StackOverflow:
 160          https://stackoverflow.com/q/4902198
 161        * My modification may be a bit slower (not too much)
 162          but it seems to be clear and reliable.
 163        '''
 164        # (1) Initial fontsize = required_font_size
 165        # (Note: fontsize [in fontsize units]
 166        # (is just APPROXIMATELY equal to required fontsize [in pixels].
 167        font_size = round(required_font_size_in_pixels)
 168        
 169        # (2) Initialize draw_object + font_object and calculate font height
 170        # (Note1: draw_object is needed for the correct font height calculation
 171        # (Note2: we calculate the font height for a model text - here: cap 'M'
 172        draw_object = ImageDraw.Draw(self.img)
 173        font_object = ImageFont.truetype(font_name, font_size)
 174        font_height = MyImage.font_height_pix(draw_object, font_object) 
 175        # (2a) Current font_height > required_font_size_in_pixels, decrease...
 176        while font_height > required_font_size_in_pixels:
 177            font_size -= 1
 178            font_object = ImageFont.truetype(font_name, font_size)
 179            font_height = MyImage.font_height_pix(draw_object, font_object)
 180        # (2b) Current font_height > required_font_size_in_pixels, increase...
 181        while font_height < required_font_size_in_pixels:
 182            font_size += 1
 183            font_object = ImageFont.truetype(font_name, font_size)
 184            font_height = MyImage.font_height_pix(draw_object, font_object)
 185        
 186        # (3) Return font size (for given font_name, in font_name units)
 187        return(font_size)

 194        return(text_height)

 197    def to_gray(self, itype='8bit'):
 198        '''
 199        Convert image to grayscale.
 200
 201        Parameters
 202        ----------
 203        itype : str, optional, '8bit' or '16bit', default is '8bit'
 204            The image is converted either to 8bit or 16bit grayscale.
 205
 206        Returns
 207        -------
 208        None
 209            The output is saved in self.img.
 210            
 211        Technical notes
 212        ---------------
 213        * Conversion to 16-bit grayscale can be tricky.
 214        * RGB -> 16-bit grayscale conversion is not supported now.
 215        * 8-bit-gray -> 16-bit-gray changes dtype to 16bit, but not values.
 216        '''
 217        if itype == '8bit':
 218            # Conversion to 8-bit grayscale (the default) should be fine.
 219            self.img = self.img.convert('L')
 220        elif itype == '16bit':
 221            # Conversion to 16-bit grayscale can be tricky - see the docstring.
 222            self.img = self.img.convert('I;16')
 223        else:
 224            print('Unknown image type when converting to grayscale!')
 225            print('The original image was not changed.')

 336    def resize(self, width=None, height=None, resample=None):
 337        '''
 338        Resize image to new width or height, keeping the aspect ratio.
 339    
 340        Parameters
 341        ----------
 342        width : int, optional, default is None
 343            The required width of the resized image.
 344            The height will be calculated
 345            so that the aspect ratio was preserved.
 346        height : int, optional, default is None
 347            The required height of the resized image.
 348            The width will be calculated
 349            so that the aspect ratio was preserved.
 350        resample : int or None
 351            If None, default resampling filter is used (usually Ok).
 352            It is also possible to define a specific resampling filter.
 353            For sharp images or plots we may prefer no resampling and 
 354            we can use *resampling = 0 = Image.Resampling.NEAREST*.
 355            More: https://pillow.readthedocs.io -> Reference -> Image -> resize
 356            
 357        Returns
 358        -------
 359        None
 360            The output is saved in *self.img*
 361        '''
 362        # (0) Calculate AR = aspect_ratio
 363        aspect_ratio = self.height/self.width
 364        
 365        # (1) Calculate new dimensions while keeping aspect_ratio
 366        if width is not None:     # New dimensions if the width is given
 367            new_width  = round(width)
 368            new_height = round(width * aspect_ratio)
 369        elif height is not None:  # New dimensions if the height is given
 370            new_height = round(height)
 371            new_width  = round(height * 1/aspect_ratio)
 372        else:
 373            print('Image resize: width or height was not given - no action!')
 374            return
 375        
 376        # (2) Resize the image
 377        self.img = self.img.resize(
 378            (new_width, new_height), resample=resample)

 431    def scalebar(self, pixsize, F=None, **kwargs):
 432        '''
 433        Insert a scalebar in the lower right corner of the image.
 434
 435        Parameters
 436        ----------
 437        pixsize : str
 438            Description how to determine pixel size.
 439            Pixel size is needed to calculate the scalebar length.
 440            See *Example* section below to see available options.
 441        F : float, optional, the default is None
 442            Multiplication coefficient/factor that changes the scalebar size.
 443            If F = 1.2, then all scalebar parameters are enlarged 1.2 times.
 444        kwargs : list of keyword arguments
 445            Allowed keyword arguments are:
 446            color, bcolor, position, stripes, messages.
 447            See section *List of allowed kwargs* below for more info. 
 448            
 449        Returns
 450        -------
 451        None
 452            The scalebar is drawn directly to *self.img*.
 453            
 454        Example
 455        -------
 456        >>> # Four basic ways how to insert a scalebar in an image
 457        >>> # (model example; in real life we use just one of the ways
 458        >>>
 459        >>> # (0) Import api + read image
 460        >>> import myimage.api as mi
 461        >>> img = mi.MyImage('../IMG/image123_20kx.bmp')
 462        >>> 
 463        >>> # (1) Pixel size from real width of image = 100um
 464        >>> img.scalebar('rwi,100um')
 465        >>>
 466        >>> # (2) Pixel size from a known length in image => 50 nm = 202 pixels
 467        >>> img.scalebar('knl,50nm,202')
 468        >>> 
 469        >>> # (3) Pixel size from known magnification
 470        >>> # (note: this can be done only for calibrated microscope
 471        >>> # (calibrated microscopes => myimg.settings.MicCalibrations
 472        >>> # (3a) magnification deduced from last part of image name
 473        >>> # (note: mag = everything between last underscore and suffix
 474        >>> # (in this example we have: ../IMG/image123_20kx.bmp => mag = 20kx
 475        >>> img.scalebar('mag,TecnaiVeleta')
 476        >>> # (3b) magnification inserted directly
 477        >>> # (note: mag can be something like 20kx, 20k, 20000x, 20000
 478        >>> img.scalebar('mag,TecnaiVeleta,20kx')
 479        >>>
 480        >>> # (4) Pixel size from accompanying text file
 481        >>> # (note: some microscopes save images + descriptive txt files
 482        >>> # (the format of text file must be described somehow
 483        >>> # (description of text files => myimg.settings.MicDescriptionFiles
 484        >>> img.scalebar('txt,MAIA3')
 485        >>>

 609class Montage:
 610    '''
 611    Class defining Montage objects.
 612    
 613    * Montage object = a rectangular multi-image. 
 614    * See __init__ for more information about initial object parameters.
 615    * More help: https://mirekslouf.github.io/myimg/docs/pdoc.html/myimg.html
 616    '''
 617    
 618    def __init__(self, images, itype,
 619                 grid=None, padding=0, fill='white', crop=True, rescale=None):
 620        '''
 621        Initialize Montage object.
 622
 623        Parameters
 624        ----------
 625        images : list of images (arrays or str or path-like or MyImage objects)
 626            The list of images from which the montage will be created.
 627            If {images} list consists of arrays,
 628            we assume that these arrays are the direct input to montage
 629            (more precisely: the direct input to skimage.util.montage method).
 630            If {images} list contains of strings or path-like objects,
 631            we assume that these are filenames of images
 632            that should be read as arrays.
 633            If {images} lists contains MyImage objecs,
 634            we use MyImage objects to create the final montage.
 635        itype : type of images/arrays ('gray' or 'rgb' or 'rgba')
 636            The type of input/output images/arrays.
 637            If itype='gray',
 638            then the input/output are converted to grayscale.
 639            If itype='rgb' or 'rgba'
 640            then the input/output are treated as RGB or RGBA images/arrays.
 641        grid : tuple of two integers (number-of-rows, number-of-cols)
 642            This argument is an equivalent of
 643            *grid_shape* argument in skimage.util.montage function.
 644            It defines the number-of-rows and number-of-cols of the montage.
 645            Note: If grid is None, it defaults to a suitable square grid.
 646        padding : int; the default is 0
 647            This argument is an equivalent of
 648            *padding_width* argument in skimage.util.montage function.
 649            It defines the distance between the images/arrays of the montage.
 650        fill : str or int or tuple/list/array; the default is 'white'
 651            This argument is a (slightly extended) equivalent of 
 652            *fill* argument in skimage.util.montage function.
 653            It defines the color between the images/arrays.
 654            If fill='white' or fill='black',
 655            the color among the images/arrays is white or black.
 656            It can also be an integer value (for grayscale images)
 657            or a three-value tuple/list/array (for RGB images);
 658            in such a case, it defines the exact R,G,B color among the images.
 659        crop : bool; the default is True
 660            If crop=True, the outer padding is decreased to 1/2*padding.
 661            This makes the montages nicer (like the outputs from ImageMagick).
 662        rescale : float; the default is None
 663            If *rescale* is not None, then the original size
 664            of all input images/arrays is multiplied by *rescale*.
 665            Example: If *rescale*=1/2, then the origina size
 666            of all input images/arrays is halved (reduced by 50%).
 667            
 668        Returns
 669        -------
 670        Montage object = multi-image/montage of *images*.
 671        The montage object can be shown (Montage.show) or saved (Montage.save).
 672        
 673        Technical notes
 674        ---------------
 675        * Only 'gray', 'rgb', and 'rgba' standard formats are supported.
 676          If an image has some non-standard format,
 677          it can be read and converted using a sister MyImage class
 678          (methods MyImage.to_gray, MyImage.to_rgb, MyImage.to_rgba).
 679        * The user does not have to differentiate 'rgb' and 'rgba' images.
 680          It is enough to specify 'rgb' for color images
 681          and if the images are 'rgba', the program can work with them.
 682        '''
 683        
 684        # Get basic arguments/properties
 685        self.images  = images
 686        self.itype   = itype
 687        self.grid    = grid
 688        self.padding = padding
 689        self.fill    = fill
 690        self.crop    = crop
 691        self.rescale = rescale
 692        
 693        # Process self.images
 694        # 1) Convert all images to np.arrays.
 695        # 2) Check if the np.arrays have the same size,
 696        #    i.e. the same dimensions, including the number of channels.
 697        #    (channels ~ colors + transparency; usually the last dim/axis
 698        # 3) Rescale images/arrays if requested.
 699        #    (the images/arrays are rescaled if self.rescale != None
 700        self.process_images()
 701        
 702        # Process self.itypes
 703        # 1) Check image tepes
 704        #    (the only allowed image types are 'rgb' and 'gray'.
 705        # 2) Define additiona parameters for montage:
 706        #    (the additional parameters will be saved within self object
 707        self.check_image_types()
 708        
 709        # Rescale images if requested
 710        # (rescaleing can be performed AFTER...
 711        # (...self.process_images => because images are converted to arrays
 712        # (...self.check_image_types => because this sets additional params:
 713        # (   self.montage_channel_axis
 714        # (   self.montage_number_of_channels
 715        if self.rescale is not None: self.rescale_images()
 716        
 717        # Adjust white/black colors according to image type and no-of-channels
 718        if (self.itype == 'rgb') or (self.itype == 'rgba'): 
 719            image_black = (0,)   * self.montage_number_of_channels
 720            image_white = (255,) * self.montage_number_of_channels 
 721        elif self.itype == 'gray':
 722            image_black = 0
 723            image_white = 255
 724                
 725        # Analyze fill = fill_color and adjust acc.to image_type
 726        match fill:
 727            case 'white':
 728                self.fill = image_white
 729            case 'black':
 730                self.fill = image_black
 731    
 732        # Create final montage
 733        self.montage = ski.util.montage(
 734            arr_in=self.images,
 735            grid_shape=self.grid, fill=self.fill,
 736            padding_width=self.padding, channel_axis=self.montage_channel_axis)
 737        
 738        # Crop the montage (= reduce the outer padding size to 1/2*padding)
 739        if self.padding and self.crop:
 740            my_crop = round(self.padding / 2)
 741            self.montage = self.montage[my_crop:-my_crop, my_crop:-my_crop]

 744    def process_images(self):
 745        '''
 746        Check and prepare images before creating montage.
 747        
 748       * The images used for montage
 749         should be convertible to arrays with the same type and size.
 750       * This method goes through all images in self.images
 751         and converts them to arrays.
 752       * If the converted images/arrays have the same dimensions(=type),
 753         then everything is Ok
 754         and we cen proceed.
 755       * If the conerted images/arrays do not have the same dimension(=type),
 756         then this methods ends with an error
 757         and the user has to adjust images BEFORE calling the montage method.
 758       
 759        Returns
 760        -------
 761        None
 762            If everything is Ok,
 763            the checked images are read and saved as arrays in the self.images.
 764            If the images are not of the same type
 765            this method prints an error and the program quits.
 766        '''
 767        
 768        # (0) Prepare variables
 769        img_sizes = []
 770        
 771        # (1) Go through the images (self.images) to create montage array.
 772        for i,image in enumerate(self.images):
 773            # Case 1: NumPy array => just 
 774            if type(image) == np.ndarray:
 775                # self.images is not changed = we already have an array
 776                # just save image size
 777                img_sizes.append(image.shape)
 778            # Case 2: MyImage object => convert to array and append 
 779            elif 'MyImage' in str(type(image)):
 780                # self.images[i] is changed => np.array instead of MyImg object
 781                self.images[i] = np.array(image.img)
 782                # save image size
 783                img_sizes.append(self.images[i].shape)
 784            # Case 3: None object => append None
 785            # TODO: Later, None should be exchanged for suitable empty array.
 786            elif type(image) == None:
 787                pass
 788            # Case 4: Nothing of the above => we suppose it is a filename.
 789            # TODO: Consider exceptions etc...
 790            else:
 791                # self.images[i] is changed => we read a file to an array
 792                if 'rgb' in self.itype:
 793                    # Read image in the default way (as_gray=False)
 794                    self.images[i] = ski.io.imread(image, as_gray=False)
 795                elif 'gray' in self.itype:
 796                    # Read image as grayscale (as_grayscale=True)
 797                    self.images[i] = ski.io.imread(image, as_gray=True)
 798                # Save image size
 799                img_sizes.append(self.images[i].shape)
 800                
 801        # (2) Additional adjustment for all grayscale images
 802        # => if itype = 'gray', then normalize image to 8bit grayscale.
 803        # (3-value gray-imgs normalized to 1, 8bit to 255, 16bit to 65535
 804        # (moreover, we can have a combination of various grayscales
 805        # (to get reproducible results, we ALWA&S normalize to 8-bit gray
 806        for i,image in enumerate(self.images):
 807            if 'gray' in self.itype:
 808                arr_max = np.max(self.images[i])
 809                self.images[i] = np.round(
 810                    self.images[i]/arr_max * 255).astype(np.uint8)
 811        
 812        # (3) Final check if all images are of the same size
 813        # Trick: All-list-elements-are-the-same
 814        # if the number of the occurrences of the first elements
 815        # equals to the total number of elements in the list.
 816        all_the_same = img_sizes.count(img_sizes[0]) == len(img_sizes)
 817        # Now the final check
 818        if all_the_same:
 819            return(self.images)

 822    def check_image_types(self):
 823        '''
 824        Check image/array types and prepare additional parametrs for montage.
 825        
 826        * The only allowed image types are 'rgb' and 'gray'.
 827        * Other image types => print error message and exit program.
 828        * For each image type we have to set two additional montage params:
 829            - self.montage_channel_axis => the last axis of the array
 830            - self.montage_number_of_channels => gray ~ 1, rgb ~ 3, rgba ~ 4
 831        
 832        Returns
 833        -------
 834        None
 835            The additional montage parameters are are saved in
 836            self.montage_channel_axes and self.montage_number_of_channels.
 837            These parameters are needed
 838            for correct treatement of rgb/gray images/arrays.
 839        '''
 840        
 841        if (self.itype == 'rgb') or (self.itype == 'rgba'):
 842            # RGB and RGBA images = 3D-arrays
 843            # the last axis = channel axis = color channels = axis 4 => index 3
 844            # the number of channels = either 3 (RGB) or 4 (RGBA)
 845            self.montage_channel_axis = 3
 846            self.montage_number_of_channels = self.images[0].shape[-1]
 847        elif self.itype == 'gray':
 848            # Grayscale images = 2D-arrays
 849            # (no channel axis = just one channel, grayscale value
 850            # (all grayscale images are read by ski.io.imread as 1-value gray
 851            self.montage_channel_axis = None
 852            self.montage_number_of_channels = 1
 853        else:
 854            # Other image formats are not supported at the moment.
 855            print(f'Unknown image type: [{self.itype}]!')
 856            print("Allowed image types are 'gray', 'rgb' or 'rgba'.")
 857            sys.exit()

1099        if input1 == float or input1 == int:
1100            # 1st input is a float (number) => 2nd input should be str (units)
1101            if input2 == str:
1102                self.number = number
1103                self.units  = units
1104            else:
1105                print('Error in defining NumberWithUnits object!')

1123                ''',

1125                flags = re.VERBOSE)

1127            self.number = float(number)
1128            self.units  = units
1129        elif input1 == NumberWithUnits:

137        '''
138        Get font size (in fontsize units)
139        corresponding to *required_font_size_in_pixels*.
140
141        Parameters
142        ----------
143        fontname : str
144            Name of the TrueType font to use.
145            Example (working in Windows): font_name='timesbd.ttf'
146        required_font_size_in_pixels : float
147            Required font size in pixels.
148
149        Returns
150        -------
151        font_size : int
152            Final font size (in font units).
153            If the returned *font_size* (in fontsize units)
154            is applied to font with given *fontname*,
155            then the height of the font (in pixels units)
156            will correspond to *required_font_size_in_pixels* argument.
157            
158        Technical notes
159        ---------------
160        * This function is a modified recipe from StackOverflow:
161          https://stackoverflow.com/q/4902198
162        * My modification may be a bit slower (not too much)
163          but it seems to be clear and reliable.
164        '''
165        # (1) Initial fontsize = required_font_size
166        # (Note: fontsize [in fontsize units]
167        # (is just APPROXIMATELY equal to required fontsize [in pixels].
168        font_size = round(required_font_size_in_pixels)
169        
170        # (2) Initialize draw_object + font_object and calculate font height
171        # (Note1: draw_object is needed for the correct font height calculation
172        # (Note2: we calculate the font height for a model text - here: cap 'M'
173        draw_object = ImageDraw.Draw(self.img)
174        font_object = ImageFont.truetype(font_name, font_size)
175        font_height = MyImage.font_height_pix(draw_object, font_object) 
176        # (2a) Current font_height > required_font_size_in_pixels, decrease...
177        while font_height > required_font_size_in_pixels:
178            font_size -= 1
179            font_object = ImageFont.truetype(font_name, font_size)
180            font_height = MyImage.font_height_pix(draw_object, font_object)
181        # (2b) Current font_height > required_font_size_in_pixels, increase...
182        while font_height < required_font_size_in_pixels:
183            font_size += 1
184            font_object = ImageFont.truetype(font_name, font_size)
185            font_height = MyImage.font_height_pix(draw_object, font_object)
186        
187        # (3) Return font size (for given font_name, in font_name units)
188        return(font_size)

  • The objet defines a scalebar: its lenght (number,units) and length-in-pixels (pixels).

Object initialiation

>>> # Three basic ways how to initialize a ScaleWithUnits object
>>> from myimg.nwu import ScaleWithUnits
>>> swu1 = ScaleWithUnits('1.2um', pixels=100)
>>> swu2 = ScaleWithUnits(number=1.2, units='um', pixels=100)
>>> swu3 = ScaleWithUnits(number_with_units_object, pixels=100)

List of object properties

  • number = number/numeric value
  • units = units corresponding to number
  • pixels = pixels corresponding to number-with-units
  • _units_description: myimg.settings.Units subclass, private property
